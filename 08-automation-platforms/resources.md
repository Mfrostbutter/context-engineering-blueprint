# Automation Platform Resources

---

## n8n

**Official docs:** https://docs.n8n.io
Excellent documentation. Start with the "Getting started" section and the "Courses" section.

**Official YouTube:** https://www.youtube.com/@n8n-io
Workflow tutorials, new feature releases, AI integration walkthroughs.

**Automate All The Things:** https://www.youtube.com/@automateallthethings
Community channel focused on real-world n8n workflows. Great for seeing what's actually possible.

**Community forum:** https://community.n8n.io
Active forum. Post a question, get an answer. Also searchable — most common problems are already answered.

**Template library:** https://n8n.io/workflows
Hundreds of pre-built workflows you can import and modify. Good starting point for any common use case.

**Free beginner course:** https://docs.n8n.io/courses/
n8n's own course. Walks through building real workflows from scratch.

## Make

**Official docs:** https://www.make.com/en/help
Comprehensive reference documentation.

**YouTube:** Search "Make (Integromat) tutorial" — several good community channels
Recommended: "Iridium Tech" and "Mike Socha" on YouTube for Make tutorials.

**Community:** https://community.make.com

## Zapier

**Official docs:** https://zapier.com/learn
Good beginner guides and use-case tutorials.

**YouTube:** Search "Zapier tutorial" — Zapier's own channel is solid.

**Templates:** https://zapier.com/apps/integrations
Massive library of pre-built Zaps for every app combination.

## General automation / AI workflow learning

**No Code MBA:** https://www.nocodemba.com
Courses on building with no-code tools including automation platforms. Has a paid tier, but the free content is useful.

**Lenny's Newsletter:** https://www.lennysnewsletter.com
Frequently covers AI + automation for product and ops teams. Free tier is excellent.

**Ben's Bites:** https://www.bensbites.com
Daily newsletter covering practical AI tools and automation. Good for staying current.

## Prompt for building AI-powered workflows

Once you're comfortable with your automation platform, use this prompt in Claude Code or Claude.ai to help design a workflow:

```
I want to build an automation workflow. Here's what I want it to do:
[describe your workflow in plain English]

I'm using [n8n / Make / Zapier]. 

Please:
1. Break down the workflow into steps
2. Identify which node/module I need for each step
3. Identify any credentials I'll need to set up
4. Flag any potential issues or limitations I should know about
5. Suggest any Claude AI steps that could add value to this workflow
```
